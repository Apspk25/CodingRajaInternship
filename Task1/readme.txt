# Text Data for Sentiment Analysis
# Link to download - https://www.kaggle.com/datasets/akashharh/text-data
# I've downloaded it and saved it as 'sentiment_analysis.csv'.

This dataset contains social media posts labeled with sentiments (positive, negative, or neutral). It can be used for training and evaluating sentiment analysis models.

## Dataset Description

The dataset includes the following columns:

- `textID`: Identifier for each text entry.
- `text`: The full text of the social media post.
- `selected_text`: A portion of the text considered relevant.
- `sentiment`: The sentiment label (positive, negative, neutral).
- `Time of Tweet`: The time when the tweet was posted.
- `Age of User`: The age group of the user.
- `Country`: The country of the user.
- `Population -2020`: The population of the user's country in 2020.
- `Land Area (Km²)`: The land area of the user's country.
- `Density (P/Km²)`: The population density of the user's country.
