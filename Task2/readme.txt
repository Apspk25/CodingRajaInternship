Fruits-360 Dataset (https://www.kaggle.com/datasets/moltean/fruits/data)
This dataset contains images of various fruits, originally sourced from Kaggle. It includes both training and test datasets for building and evaluating machine learning models for fruit recognition.

Dataset Contents
Training Data: Contains images used for training machine learning models.
Test Data: Contains images used to evaluate the trained models.
Dataset Details
Total Classes: 131
Dataset Size:
Training: 67,692 images
Test: 22,688 images
Image Size: Resized to 100x100 pixels
Dataset Structure
The dataset is structured as follows:

fruits-360_dataset/
│
├── Training/
│   ├── Apple Braeburn/
│   ├── Apple Golden 1/
│   ├── ...
│   └── Walnut/
│
└── Test/
    ├── Apple Braeburn/
    ├── Apple Golden 1/
    ├── ...
    └── Walnut/
Each fruit category contains multiple images of that fruit.

Usage
Use this dataset to train and evaluate machine learning models for fruit recognition tasks.
Preprocessing, such as resizing and normalization, may be required depending on the model architecture used.
